global-jjb v0.1.0 Release Notes
===============================

This is a general announcement that Global JJB v0.1.0 is now released [0]. This
is our first released version and provides the following:

* jjb-merge job
* jjb-verify job

Detailed instructions on how to deploy it into a ci-management repo can be
found here [1] and we will be deploying this to OpenDaylight via this patch [2]
as a first example.
